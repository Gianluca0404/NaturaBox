# NaturaBox - E-commerce for Pet Products

NaturaBox is a simple e-commerce platform designed to sell pet products. The site allows customers to browse, search, and purchase various items for their pets, providing a seamless shopping experience. It includes essential features like user authentication (registration and login), a shopping cart system, and an order checkout process.

## Features

* **User Registration & Login:** Allows customers to create an account, log in, and manage their personal information.
* **Product Catalog:** Users can browse through a wide variety of pet products, each with detailed descriptions and pricing.
* **Checkout:** A simple and straightforward checkout page where users finalize their orders.
* **Profile Management:** Registered users can view and edit their profile details.
* **Live Search:** Users can quickly search for products with live search functionality.

## Technologies Used

* **Frontend:** HTML, CSS, JavaScript for building the user interface and handling dynamic interactions.
* **Backend:** PHP for server-side logic and handling requests such as user authentication, cart actions, and order processing.
* **Database:** MySQL for storing product, user, and order data.

## Getting Started

To get started with **NaturaBox** locally, you will need a local server environment (e.g., XAMPP, WAMP, or MAMP) and a MySQL database. After setting up the server, clone the repository and set up the database as specified in the project.

1. **Install Dependencies:** Set up the local server and configure the database connection.
2. **Run the Server:** Start your local server and open the application in a browser at `http://localhost/naturabox` to begin using the e-commerce platform.
